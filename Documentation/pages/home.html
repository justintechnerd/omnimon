About the project.

Omnimon is an open-source virtual pet simulator designed for low-spec systems. With a user-friendly editor, you can create and customize your own virtual pet experience.

This document contains all the information you need to get started with Omnimon, including installation instructions, user guides, and troubleshooting tips.

This is an in-development fan-project with no connection to actual vpet manufacturers.
